Automação com Playwright

Este projeto é dedicado ao estudo e prática de automação de testes utilizando o framework Playwright, explorando testes modernos para a web.

Objetivos do Projeto

  • Explorar testes cross-browser (Chromium, Firefox, WebKit).
  • Implementar o padrão Page Object Model (POM) para organização.
  • Execução de testes paralelos e geração de relatórios detalhados.

Destaques

Foco em reduzir a instabilidade (flakiness) dos testes e garantir uma execução rápida e confiável em pipelines de CI/CD.
