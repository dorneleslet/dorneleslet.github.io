Lee.dev Voltar

Sistema de Gestão de Clientes, Serviços, Caixa e Estoque

Este sistema foi desenvolvido para atender às necessidades reais de um salão de beleza, focando na organização e eficiência operacional. Atualmente em uso no salão de beleza e em melhorias e adição de novas funcionalidades.


Tecnologias Utilizadas

  • Backend: Python
  • Frontend: JavaScript, HTML, CSS

Qualidade e Testes (QA)

Como o foco da minha carreira é voltada para área de QA, este projeto foi o cenário principal para aplicar:

  • Casos de Teste: Criação de casos de teste com todos os fluxos.
  • Testes Manuais: Execução de testes funcionais e exploratórios.
  • Gestão de Bugs: Identificação e correção de falhas críticas.
  • Testes Automatizados: Implementação de fluxos E2E com Cypress.
  • Testes de API: Validação de endpoints utilizando Postman.
  • Testes de Regressão: Implementação de todos os fluxos listados pelos casos de teste com Cypress.

Funcionalidades

O sistema conta com controle de fluxo de caixa com histórico e registros de recibos, cadastro detalhado de clientes com fichas técnicas de procedimentos, gerenciamento de estoque com histórico de entrada e saída de produtos.
